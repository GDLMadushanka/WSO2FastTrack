    • Method Detail

      • isLoaded

        boolean isLoaded()
        Check whether the DataHandler has already been loaded. A return value of true means that a call to getDataHandler() will not block or will retrieve the DataHandler without overhead. Note the return value of this method for a given instance of this class may change over time due to events other than a call to getDataHandler() on the same instance. E.g. a call to getDataHandler() on one instance may change the return value of the method on another instance (because the DataHandler objects can only be loaded in a certain sequence).
        Returns:
        true if the DataHandler has already been loaded; false otherwise
