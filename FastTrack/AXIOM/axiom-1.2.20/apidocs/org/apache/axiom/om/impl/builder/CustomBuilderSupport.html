Interface CustomBuilderSupport

  • All Known Implementing Classes:
    MTOMStAXSOAPModelBuilder, StAXBuilder, StAXOMBuilder, StAXSOAPModelBuilder, XOPAwareStAXOMBuilder

    public interface CustomBuilderSupport
    Interface allowing registration of a custom builder. This is a semi-public API used by Axis2; it is not meant for general consumption.
