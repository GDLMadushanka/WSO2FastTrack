      • getUnmarshallerAdapter

        public static UnmarshallerAdapter getUnmarshallerAdapter(XMLStreamReader reader)
        Create an adapter that enables an Unmarshaller to perform XOP/MTOM processing on an XMLStreamReader that exposes the DataHandlerReader extension. This method should be used in the following way:
        1. Create the Unmarshaller.
        2. Call this method with the XMLStreamReader to be unmarshalled.
        3. Configure the Unmarshaller with the AttachmentUnmarshaller returned by UnmarshallerAdapter.getAttachmentUnmarshaller().
        4. Call Unmarshaller.unmarshal(XMLStreamReader) or Unmarshaller.unmarshal(XMLStreamReader, Class), passing the XMLStreamReader returned by UnmarshallerAdapter.getReader() as parameter.
        Parameters:
        reader - an XMLStreamReader that may expose the DataHandlerReader extension
        Returns:
        the adapter
